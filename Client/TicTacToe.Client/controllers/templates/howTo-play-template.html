Create game

To create a game you need to go to create page and click the button create. In future versions of the application will be added options to give a name of the game, to be a private (that have a password) and shareable link to invite friend.

Join game

Currently there are a option to join in the first available game on random choice, but this will be fixed in next updates of the application. The target is to have a join button on each game that waits for opponent. However you use only joining options for now at join page

In game

When you are in the game, first check the status. It will tell you what action to do. If there are no another opponent in the game you should wait. If it's your turn - click on the empty tile you want to place mark and then click play button else should wait until opponent plays. Pop up messages are showing from top-right of the screen if you make mistake.

- All operations needs a registered account.
